Data‑Driven DJ: Inter‑Country Wedding Playlist Insights
A cross‑regional Spotify playlist analysis exploring music trends in Zimbabwe, Ghana, and Nigeria to support culturally balanced and inclusive inter‑country wedding playlists.

Overview
This project analyzes up to 50 curated Spotify playlists per country to identify:

Recurring tracks within each region

Cross‑border hits shared between countries

Cultural patterns useful for wedding and event playlist design

All extracted tracks were compiled into structured dataframes, enriched with genre information, and exported as CSV files for further use.

Key Findings
Nigeria had the largest playlist dataset, producing the highest number of recurring tracks.

Ghana showed moderate playlist volume with notable overlap with Nigerian music.

Zimbabwe had fewer playlists, resulting in fewer recurring tracks but still essential for cultural representation.

No recurring tracks were shared between Zimbabwe and the West African countries, highlighting distinct regional music identities.

Application for Weddings
Insights from the data help DJs create:

Balanced multicultural playlists

Smooth transitions using cross‑border hits

Fair cultural representation, especially in Zimbabwean–Nigerian weddings

Generational balance when combined with release‑year metadata (future enhancement)

Limitations
BPM data could not be extracted from Spotify, limiting tempo‑based analysis.

Playlist availability varied significantly by country.

Only three African countries were included due to time constraints.

Future Work
Add BPM and release‑year data for deeper musical profiling.

Expand analysis to more African regions (East, North, Central, Southern).

Group countries by region to identify continent‑wide trends.
